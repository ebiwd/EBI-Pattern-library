--- layout: pattern title: Live filtering categories: patterns tags: - typography - search description: If you have mutliple rows of data you'd like the use to be able to quickly sift through (perhaps combining with table sorting) we bundle the aptly named LiveFilter plugin. It is not loaded by default and you'll need to load LiveFilter/js/jquery.liveFilter.js. ---

There are many such JS plugins available, but we found this one to work well and do just enough and easily modldable to our needs.

Bonus notes
  • The sample input has had .clearable added so clear button shows after text has been added.
  • We hint about the action that will be performed by placing a search icon to the left of the box, it does not however act as a button as the form is automatically updated on keypress.
