The fonts.css stylesheet automatically loads a range of icon fonts that you can use.

Icons are added automatically through CSS using the :before feature in CSS, so they will appear before whatever piece of text you associate them with. To make this association, you need to include particular class and data-icon attributes in the relevant HTML element: e.g. <h4><span class="icon icon-generic" data-icon="4"></span> Travel by plane</h4>.

Travel by plane

The web browser will read the data-icon parameter and use the icon pictogram corresponding to "4".

Note this method is now widely supported. We do not recommend using PNG images as they offer inferior performance and inferior support for high resolution screens.

This requires JavaScript to work as designed. jQuery and the Foundation JS library are not required. Strictly, JavaScript is not needed, but this feature will not execture if the user has JavaScript disabled.
