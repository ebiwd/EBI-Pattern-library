You can also add a caption with .caption.

For images inserted inside text areas, you can use the CSS class of .thumbnail to add subtle padding and spacing.

You can further combine with .float-left or .float-right for a nice treatment, and size with, say, .columns .medium-4 for a responsive effect.

Images will dynamically scale down to fit the container, but will not scale above their actual size.

Image overlays

Alternatively, you can use a .with-overlay for a darkened effect with a text overlay.

Notes:

  • Links: when the caption is a link the text automatically takes on a large font face and arrow. This style is best used as an image button to tease and link further content.
  • Image height: this method expects your images with have shared aspect ratios or the same width and height. If you are unable to predict or control the sizes, you can use the equilization technique.
