The default font used throughout the page is Helvetica Neue, and is implement through a web font loaded by fonts.css. This is done to reflect EMBL-wide branding.

Sizing of default text is done with rem; equivlent to 16px on 24px leading, or 12pt on 16pt leading.

For large narrative text areas, consider the CSS class .serif to use the "EB Garamond" font, which is also loaded by the font stylesheet. You can switch back to sans-serif styling with the class .sans-serif.
